Question-1: What are the different ways to select an element in the DOM?

There are some selectors given to below:

  1. getElementById() : That can select html element by id name.
  2. getElementsByClassName() : That can select html element by class name.
  3. getElementsByTagName() : That can select html element by tag name.
  4. querySelector() : QuerySelector select first element that mached css selector.
  5. querySelectorAll() : QuerySelectorAll that can select all element that match a css selector.
  6. childNodes : Selects all child nodes of a given element.
  7. firstChild / lastChild : select firstChild and lastChild of parent element
  8. parentNode : Selects the parent node of a given element.
  9. closest() : Selects the nearest ancestor of an element that matches the provided selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Different between innerHTML, innerText, and textContent given to show:

innerHTML : It returns or sets the HTML content of an element (including tags).

innerText : That can returns of innerText value.

textContent : It returns or sets the text content of an element, including text from all child nodes, without rendering HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM:
Event delegation is a technique in JavaScript where a single event listener is attached to a parent element to handle events on its child elements through event bubbling.

Question-4: What is event bubbling in the DOM?

What is event bubbling in the DOM?
Event bubbling is a mechanism in the DOM where an event, such as a click, starts from the target element and "bubbles up" to its parent elements, allowing the parent elements to also respond to the event.
